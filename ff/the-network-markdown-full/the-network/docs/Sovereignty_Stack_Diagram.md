# Sovereignty Stack (Textual Diagram)
Identity → Governance → Economy → Application → Recognition → Legal → Continuity → Deployment

- Governance: Assembly/Council/Arbitration  
- Economy: NWC (PoC) + IBZ (regional)  
- Apps: *.the.network / *.eivissa.network  
- Recognition: on-chain registry → Wikidata  
- Legal: JSON-LD treaties  
- Continuity: snapshots, reconstitution
